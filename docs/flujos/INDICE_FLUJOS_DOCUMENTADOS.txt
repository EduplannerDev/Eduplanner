===============================================================================
                              CÓMO CONTRIBUIR
===============================================================================

PARA AGREGAR NUEVA DOCUMENTACIÓN:
1. Crea un archivo .txt en /docs/flujos/
2. Sigue la convención de nombres: COMO_[ACCION]_[MODULO].txt
3. Usa el formato estándar con secciones delimitadas por ===
4. Ejecuta el script de embeddings para vectorizar el contenido

PARA ACTUALIZAR DOCUMENTACIÓN EXISTENTE:
1. Modifica el archivo .txt correspondiente
2. Ejecuta nuevamente el script de embeddings
3. Los embeddings se regenerarán automáticamente

CONVENCIONES DE FORMATO:
- Usar títulos descriptivos con ===
- Numerar pasos claramente
- Incluir ejemplos específicos
- Agregar sección de problemas comunes
- Incluir consejos y buenas prácticas
