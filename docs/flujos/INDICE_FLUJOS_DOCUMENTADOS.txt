===============================================================================
                        ÍNDICE DE FLUJOS DOCUMENTADOS
===============================================================================

DESCRIPCIÓN:
Este índice contiene todos los flujos de usuario documentados para la 
aplicación Eduplanner. Cada archivo explica paso a paso cómo usar las 
diferentes funcionalidades del sistema.

FECHA DE ÚLTIMA ACTUALIZACIÓN: Enero 2025

===============================================================================
                              FLUJOS PRINCIPALES
===============================================================================

1. **COMO_CREAR_PLANEACION.txt**
   - Módulo: Planeaciones
   - Flujo: Crear
   - Descripción: Cómo crear planeaciones didácticas usando IA
   - Incluye: Planeación Individual, Dosificación y CIME

2. **COMO_GESTIONAR_PLANEACIONES.txt**
   - Módulo: Planeaciones
   - Flujo: Gestionar
   - Descripción: Cómo EDITAR, ELIMINAR y EXPORTAR (PDF) planeaciones

3. **COMO_GENERAR_EXAMEN.txt**
   - Módulo: Exámenes
   - Flujo: Generar
   - Descripción: Cómo generar exámenes basados en planeaciones

4. **COMO_CREAR_PROYECTO.txt**
   - Módulo: Proyectos
   - Flujo: Crear
   - Descripción: Cómo crear proyectos educativos con IA (Fases NEM)

5. **COMO_GESTIONAR_MIS_PROYECTOS.txt**
   - Módulo: Proyectos
   - Flujo: Gestionar
   - Descripción: Gestión de proyectos e instrumentos de evaluación

6. **COMO_USAR_DOSIFICACION.txt**
   - Módulo: Dosificación
   - Flujo: Usar
   - Descripción: Distribución temporal y seguimiento curricular

7. **COMO_CREAR_PLAN_ANALITICO.txt**
   - Módulo: Plan Analítico
   - Flujo: Crear
   - Descripción: Diagnóstico, Contextualización y Codiseño

8. **COMO_CREAR_PRESENTACION_IA.txt**
   - Módulo: Presentaciones IA
   - Flujo: Crear
   - Descripción: Generación de diapositivas y exportación PPT

9. **COMO_USAR_INCIDENCIAS.txt**
   - Módulo: Incidencias
   - Flujo: Reportar
   - Descripción: Protocolos de seguridad, actas y firmas

10. **COMO_TOMAR_ASISTENCIA.txt**
   - Módulo: Asistencia
   - Flujo: Registrar
   - Descripción: Toma de lista y reportes

11. **COMO_GESTIONAR_GRUPOS_Y_ALUMNOS.txt**
    - Módulo: Grupos y Alumnos
    - Flujo: Gestionar
    - Descripción: Agregar alumnos, editar datos, crear grupos
